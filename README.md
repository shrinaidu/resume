## How to View
Open [your-username.github.io/resume](https://your-username.github.io/resume) in your browser (replace `your-username` with your actual GitHub username).

## How to Host on GitHub Pages

### Step 1: Create a GitHub Repository
1. Go to [GitHub](https://github.com) and sign in
2. Create a new repository named `resume`
3. Choose "Public" so it's accessible online
4. Click "Create repository"

### Step 2: Push Your Code to GitHub
In your terminal, navigate to your project folder and run:
```bash
git init
git add .
git commit -m "Initial commit - resume website"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/resume.git
git push -u origin main
```

### Step 3: Enable GitHub Pages
1. Go to your repository on GitHub
2. Click "Settings" (top right)
3. Scroll down to "GitHub Pages" section
4. Under "Source", select `main` branch
5. Click "Save"

Your resume will be live at: `https://YOUR-USERNAME.github.io/resume`

## Customization
